## 固件烧录驱动安装

**此文档为速通方案烧录，不涉及固件源码的编译，所以适合小白入手，大家注意文档的顺序。**

### 2. 主控烧录固件

+ 首先购买stlink烧录器 
+ 将主控板子和语音板子连接并且将语音板子的type c连接电脑或者普通充电器供电
+ 将stlink烧录器和主控板子3pin插槽连接，并且将烧录器usb连接到电脑
+ 使用ST-LINK Utility软件进行固件烧录（软件下载百度一大把，学会使用搜索引擎）
